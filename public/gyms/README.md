# Gym photos — drop your pictures here

Put **one photo per gym** in this folder. Name the file after the gym's id (below)
and it appears automatically on both the **overview list** (thumbnail) and the
**gym profile** (big photo). No code editing needed.

- Accepted types: `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`
- Recommended: landscape (roughly 16:9), at least 1200px wide.
- After adding files in dev, **restart the dev server** to pick them up
  (`npm.cmd run dev`). In production they're picked up on the next build.
